The interesting things that you don't know about C

  1. It was developed at Bell Laboratories in 1972 by Dennis Ritchie.
  2. C Language was not called C at the beginning. It has been named as C after passing many stages of evolution. Evolution of C: ALGO -> BCPL -> B -> Tradition C -> K&R C -> ANSI C -> ANSI/ISO C -> C99..
  3. C is the only programming language that exists for such a long period and still it is widely used.
  4. C is the basis of many other programming languages like C++, Java, JavaScript, Go, C#, PHP, Python, Perl, C-shell and many more
